في علم الحديث، لا يُقبل أي كلام مهما بدا صحيحاً إلا **بسند** — سلسلة إثبات موثقة ترجع إلى المصدر. «سند» يطبّق المعيار نفسه على الذكاء الاصطناعي والكود: **لا يمرّ أي ادعاء عن مشروعك إلا إذا استطاعت الخريطة المعرفية إثباته.**

يحوّل «سند» أي مشروع إلى خريطة معرفية حتمية (تحليل بنيوي للكود بلا أي نموذج ذكاء اصطناعي — مجاناً، محلياً، لا يغادر شيء جهازك)، ثم يمنح وكلاء البرمجة أربع قدرات لم تجتمع في أداة واحدة من قبل:

| الركيزة | الأمر | ماذا تفعل |
|---|---|---|
| 🛡️ **بوابة منع الهلوسة** | `sanad verify` | كل ادعاء بنيوي يقوله النموذج («س تستدعي ص»، «ع معرَّفة في الملف م») يُحاكم آلياً ضد الخريطة: **مؤكد** (مع مسار الإثبات)، **مؤكد بشكل غير مباشر**، **مجهول**، أو **مرفوض**. وفي الوضع الصارم، الادعاء المرفوض يوقف العملية كلها. |
| 🔮 **عرّاف أثر التعديل** | `sanad predict` / `check-impact` | قبل التعديل: يتنبأ بكل ملف يجب أن يتأثر ويحفظ التوقع كعقد التزام. بعد التعديل: يقارن الواقع بالعقد. أي تغيير خارج التوقع — لحظة «عدّلت شيئاً لم أفهمه» — يُرصد كـ**انحراف**. |
| 🧠 **التفكير بالخريطة** | `sanad ops` / `think` | بدل لصق الملفات في النموذج، يُصدر النموذج أوامر صغيرة (ابحث، من يستدعي، ما المسار...) ومحرك الخريطة يحسب محلياً مجاناً. كل عقدة تأخذ مرجعاً قصيراً فتكلف المتابعة توكنز معدودة. النماذج الصغيرة تحقق كالمحققين — تفكير عميق متعدد الخطوات بتكلفة شبه ثابتة، ويستحيل عليها اختراع دالة غير موجودة. |
| ⚖️ **مجلس العدسات** | `sanad council` | نفس النموذج الصغير يُستدعى كعدة أصوات — عدسة الاستخدام، عدسة الاعتماديات، عدسة المعمار، عدسة الأدلة — كل صوت يجري تحقيقه الخاص. ثم تُعقد مصالحة تدمج الأصوات، وتحكم البوابة على الجواب الموحد **وتفرض تصحيح** أي ادعاء مرفوض. |
| 🩹 **النسيج الندبي** | `sanad scars` | ينقّب سجل گيت (محلياً، بلا نموذج) فيستخرج **درجة خطورة** لكل ملف (كم مرة احتاج تعديله إصلاحاً أو تراجعاً) و**التوائم التاريخية** التي تتغير معاً. فيحذّرك العرّاف: «تلمس أ بدون ب، لكن ٨٧٪ من تعديلات أ السابقة لمست ب معها». خبرةٌ لا يملكها أعظم نموذج يدخل مشروعك بارداً. |
| 🧬 **القفل الوراثي** | `sanad lock-check` / `lock-gen` / `lock-grammar` | الخريطة تتحول إلى «حمض نووي» للمشروع: أي كود مولَّد يشير إلى واجهة غير موجودة يُرفض مع اقتراح أقرب رمز حقيقي، أو يُمنع نطقه أصلاً لحظة التوليد عبر ملفات القفل الصلب للنماذج المحلية. |
| 🌪️ **نفق الرياح** | `sanad tunnel` | يقتطع من الخريطة الحد الأدنى المتشابك مع أي تعديل، ينسخه لمشروع مصغّر معزول، ثم **يشغّله فعلياً** ويدير اختباراته الحقيقية — مع مسودة النموذج فوق أي ملف — قبل اعتماد التغيير. الواقع بدل التخيل، بصفر توكنز. |
| 🧪 **الذاكرة المعقمة** | `sanad memory` / `think --remember` | معرفة دائمة يستحيل أن تفسد: الخلاصة لا تدخل إلا إذا أثبتت البوابة كل ادعاء فيها، وأمر التعقيم يعيد محاكمة المخزون كله مع تطور الكود فيحجر ما لم يعد صحيحاً (ويبعثه إن عاد). كل تحقيق مُثبَت يجعل النظام أذكى للأبد. |

## أرقام مقيسة، لا وعود

كل الأرقام التالية من تشغيل حقيقي على خريطة هذا المستودع نفسه (أكثر من ١٠,٩٠٠ عقدة):

- **تقييم البوابة**: ١٠ حقائق صحيحة + ١٠ هلوسات مزروعة (ملفات خاطئة، دوال مخترعة، اتجاهات مقلوبة) ← **عبرت الحقائق ١٠/١٠، وأُمسكت الهلوسات ١٠/١٠**. وأثناء التطوير رفضت البوابة ادعاءً مصدره وثيقة قديمة داخل المشروع نفسه — هلوسة حقيقية أُمسكت آلياً.
- **حلقة التفكير**: أصغر نسخة جيميناي (فلاش لايت) أجابت سؤالاً معمارياً حقيقياً في ٩ خطوات، بـ**١١.٨ ألف توكن دخل / ٢١٣ خرج (~٠.٠٠٦٦ دولار)** — الجواب صحيح بالكامل ومصدَّق من البوابة ١٠/١٠.
- **المجلس**: ثلاثة أصوات، ١٥ خطوة، إجماع على سؤال أثر تعديل — **١٣/١٣ ادعاءً مؤكداً، بحوالي ٠.٠٠٨٨ دولار** إجمالاً.
- **القفل الوراثي**: هلوسات مزروعة (أسماء محرفة، مسارات استيراد ملفقة) أُمسكت والاقتراح الصحيح أولاً؛ ومع قائمة الواجهات المستخرجة من الخريطة خرجت أول مسودة **نظيفة تماماً بـ٠.٠٠٠٤ دولار**.
- **نفق الرياح**: مسودة مخربة منطقياً (صياغة سليمة وسلوك خاطئ) فضحتها **الاختبارات الحقيقية** في المشروع المقتطع خلال ثوانٍ وبصفر توكنز.
- **النسيج الندبي**: نقّب أكثر من ألف التزام في ثوانٍ؛ وكشف وحدة **٦٧٪ من تعديلاتها الثلاثين** احتاجت إصلاحاً لاحقاً.

## البداية السريعة

</div>

```bash
# ١. ابنِ الخريطة (محلي، مجاني، بلا مفتاح)
sanad update .

# ٢. حاكم جواب الذكاء الاصطناعي عن الكود
sanad verify "The `login()` function calls `validate_token()`." --strict

# ٣. حاسِب التعديلات
sanad predict "src/auth.py"        # قبل التعديل
# ... عدّل، ثم: sanad update .
sanad check-impact --strict        # انحراف = لمس ما لم يتوقعه

# ٤. دع نموذجاً صغيراً يفكر بالخريطة
sanad think "Which module is the security chokepoint?" --verify

# ٥. اعقد المجلس
sanad council "What breaks if we change sanitize_label's signature?"

# ٦. أو قُد الأوامر بنفسك / من أي وكيل (بلا مفتاح)
sanad ops --new "find auth"
sanad ops "callers n1"

# ٧. احقن خبرة السنين من سجل گيت (محلي، مجاني)
sanad scars .
sanad scars --file src/auth.py

# ٨. ولّد كوداً يستحيل أن يشير لواجهة غير موجودة
sanad lock-gen "add a helper that sanitizes labels using the project's sanitizer"
sanad lock-check draft.py

# ٩. جرّب المسودة ضد الواقع قبل اعتمادها
sanad tunnel "src/auth.py" --draft new_auth.py --at src/auth.py

# ١٠. ابنِ ذاكرة لا تفسد
sanad memory add "\`login()\` calls \`validate_token()\`."
sanad memory reverify
sanad think "..." --remember
```

يعمل مع **كلود كود، جيميناي، كيرسر، كودكس، كوبايلوت وأكثر من ١٥ وكيلاً** — الأمر `sanad install` يسجّل المهارة. ومحركات التفكير المدعومة: جيميناي، كلود، أوبن أيه آي، ديب سيك، كيمي، أولاما (محلي)، بيدروك، أزور.

## لماذا ليس البحث المتجهي وحده؟

البحث المتجهي يجلب *نصاً مشابهاً*؛ لكنه لا يستطيع أن يقول إن ادعاءً ما **كاذب**. خريطة «سند» تُبنى حتمياً من بنية الكود، فهي حقيقة مطلقة في الأسئلة البنيوية: من يستدعي من، ماذا يستورد ماذا، أين يُعرَّف كل شيء. وهذا ما يجعل **الرفض** — لا مجرد الجلب — ممكناً، والرفض هو قاتل الهلوسة.
